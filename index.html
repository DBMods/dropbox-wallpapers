Wallpapers? Yes, please!

This repo contains a number of different Dropbox-themed wallpapers in various sizes for your computer.

There are 3 different themes, two of which have a light version, making for 5 wallpapers total. Each of these wallpapers comes in a series of sizes:

  • 800 x 600
  • 1024 x 768
  • 1200 x 900
  • 1366 x 768
  • 1600 x 1200
  • 1600 x 900
  • 1920 x 1080

If none of these sizes suits your needs, there are 3 different aspect ratio PSDs (4:3, 16:9, and a custom one for 1920 x 1080).
